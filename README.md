# flutter_base

Flutter base for any Flutter project.

## Getting Started

This project has 2 flavors:
    - dev 
    - prod

To launch the project run the following
```
flutter run --flavor dev
flutter run --flavor prod
```

### Flavor doc

Two links were used to add flavors to project
    - https://blog.svenadolph.net/flutter-flavors-and-ios-schemes/
    - https://www.chwe.at/2020/10/flutter-flavors/

On Android we used kotlin flavors and on iOS we used schemes.
